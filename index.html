Variables

What do variables mean? Variables don't mean anything until you give them meaning. When you've automated your friend, it losts all its inherent human understanding. Now you need to explain to it every single letter of a word you're saying, even the spaces between the letters. You can tell your friend to understand x as y, and it will do just that. But for the sake of clarity, let's be practical. Let's tell it that my shoe size is 11. In code, it looks like this:

shoe_size = 11

Don't expect your mechanical friend to do anything. It does what it's told, and you only gave it an explanation to store in its memory. And it stores things from right to left. In the code above, shoe_size is the variable. The equal sign is an assignment operator. It's not like the equal sign in math. You're not telling your friend that shoe_size is equals to 11. You're telling it to place 11 to the variable shoe_size. If you want it to print the shoe size, not the variable, you type:

print(shoe_size)

Numeric Manipulation

You can ask your friend to divide, add, subtract, multiply, and give only the remainder of a division problem. It follows the PEMDAS order of operation, which it somehow remembered from elementary school and you may have not. As a refresher, PEMDAS stands for Parenthesis, Exponent, Multiplication, Addition, and Subtraction and calculates from left to right. Here is the list of math symbols in code:

  • ** : raises number to a power
  • * : multiply
  • / : divide
  • + : addition
  • - : subtraction
  • % : modulus, which only gives the remainder. Use it like in division.

It looks like the following when you code it into Python.

x = 2 ** 2 # raises the number to a power of 2
x = x * x # multiples x to x
x = x / 2 # divides x by 2
x = x + 2 # adds
x = x - 2 # subtracts

The pound sign tells our friend that its a comment not to be equated in the code. Play around with it. Use the print command to see the results.

String Manipulation

In this very basic, very introductory guide, my readers will at least distinguish strings from numerical variables. Yes, you can mix both and turn one into the other. The len() function allows Python to count each letter, space, and punctuation in within quotes. We can enumerate the Monty Python quote from above using the len() function.

print(len(pythonQuote))

You can also index places within the string using a number within brackets within the range of a string. This allows Python to locate the letter starting from a zero count.

print(pythonQuote[6])

Python can also splice segments of a string using brackets and a colon mark seperating the starting point from the end point.

print (pythonQuote[0:4])
